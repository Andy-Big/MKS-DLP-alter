Обновление оригинальной прошивки на плате MKS DLP на Alter 

Обновление оригинальной прошивки на прошивку Alter производится с помощью родного бутлоадера.
Для обновления распакуйте в корень флэшки содержимое архива, соответствующего версии платы:
mksdlp_1_1_alter_кс.кскс.zip - для платы версии 1.1
mksdlp_2_0_alter_кс.кскс.zip - для платы версии 2.0
(кс.кскс.zip  - версия прошивки Альтер)
Желательно предварительно отформатировать флэшку в формат ФАТ32.
После этого вставьте флэшку в USB разъем платы и включите питание. Родной загрузчик найдет на флэшке файл прошивки и начнет обновление. По завершению обновления стартует уже новая прошивка.
При первом старте прошивка Альтер обнаружит отсутствие необходимых ресурсов во внутренней памяти и автоматически начнет их копирование из определенных каталогов флэшки. После появления сообщения об окончании копирования необходимо выключить и опять включить питание платы. После этого новая прошивка должна нормально загрузиться, можно начинать работать.


Обновление прошивки Альтер на новую версию

Обновление старой версии прошивки Alter на новую происходит так же с помощью родного бутлоадера.
Для обновления распакуйте в корень флэшки содержимое архива, соответствующего версии платы:
mksdlp_1_1_alter_кс.кскс.zip - для платы версии 1.1
mksdlp_2_0_alter_кс.кскс.zip - для платы версии 2.0
(кс.кскс.zip  - версия прошивки Альтер)
Желательно предварительно отформатировать флэшку в формат ФАТ32.
После этого вставьте флэшку в USB разъем платы и включите питание. Родной загрузчик найдет на флэшке файл прошивки и начнет обновление. По завершению обновления стартует уже новая прошивка. При таком обновлении новая версия не будет автоматически копировать новые ресурсы из каталогов флэшки. Чтобы заставить ее сделать это нужно выключить питание, нажать правый верхний угол дисплея и включить питание, продолжая нажимать. После появления сообщений о копировании можно отпустить нажатие.
После появления сообщения об окончании копирования необходимо выключить и опять включить питание платы. После этого новая прошивка должна нормально загрузиться, можно начинать работать.
